# Portfolio Website (GitHub Pages)

A simple one-page portfolio website built with HTML, CSS, and JavaScript.

## 1) Customize your info
Edit the following in `index.html`:
- Your name and intro text
- Project titles/descriptions/links
- Skills
- Email / GitHub / LinkedIn links

## 2) Publish on GitHub Pages
1. Create a new GitHub repository (for example: `portfolio`).
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main` and `/ (root)`
5. Click **Save**.
6. Wait 1–3 minutes, then open your published URL.

For a personal site URL (`https://<username>.github.io`), use a repository named `<username>.github.io`.

## 3) Local preview
Open `index.html` directly in your browser.
